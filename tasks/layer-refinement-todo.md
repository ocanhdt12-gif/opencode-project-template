# Layer Refinement — Bug Fixes & Feature Requests

> Sau khi project hoàn thành, user sẽ báo bug hoặc feature mới vào layer này.
> Em sẽ brainstorm + đề xuất, sau đó tạo todo mới để implement.

## Workflow

1. **User báo** → Bug hoặc feature mới
2. **Em brainstorm** → Clarify + propose approaches
3. **User approve** → Confirm phương án
4. **Tạo todo** → Add vào section dưới
5. **Pick + implement** → Như layer khác
